## DataBase
    데이터를 저장할 수 있게 도와주는 소프트웨어(프로그램)

## 구조
    Database => Table => Data
    실제 데이터는 Table안에 있음.

## SQL(Structured Query Language)
    데이터베이스에 있는 데이터를 조회하기 위한 명령어

### EMP 테이블 
    empno : 사원 번호
    ename : 사원 이름
    job : 직책
    mgr : 사수 번호
    hiredate: 입사 날짜
    sal : 급여
    comm : 보너스
    deptno : 부서 번호
